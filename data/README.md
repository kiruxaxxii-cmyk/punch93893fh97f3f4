# Punch EBD (Единая база данных)

Основной файл SQLite: **`data/punch-ebd.db`**

Все аккаунты, ключи, промокоды, логи и чат лаунчера хранятся здесь.

- Локально путь задаётся автоматически: `data/punch-ebd.db`
- Переменные: `PUNCH_DATA_DIR`, `PUNCH_DB_PATH`
- В админке: вкладка **EBD** — статус, backup, удаление дубликатов email/логинов

Повторная регистрация на занятый email блокируется на уровне API и уникальных индексов.

Промокоды проверяются через `/api/promo/validate` и применяются при оплате тарифов.
